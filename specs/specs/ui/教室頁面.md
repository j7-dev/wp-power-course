# 教室頁面

## 描述
學員觀看課程的主要頁面，包含章節列表、影片播放器、進度追蹤。URL slug: `/classroom/{chapter_slug}`。

## 關鍵元素
- 章節導航列表（按 menu_order 排序，標記已完成章節）
- 影片播放器（VidStack，支援 Bunny HLS / YouTube / Vimeo / 自訂嵌入碼）
- 章節內容區域（power-editor / elementor 內容）
- 完成章節按鈕（切換完成/未完成狀態）
- 課程整體進度條
- 留言區塊（若 enable_comment = yes）
- 浮水印覆蓋層（若 pc_watermark_qty > 0）
- 手機端固定影片/Tab（若 fix_video_and_tabs_mobile = yes）

## 線性觀看模式（enable_linear_mode = yes 時）

### 章節導航列表
- 被鎖定章節在標題旁顯示 🔒 鎖頭圖示（取代原有的進度圖示）
- 已解鎖章節顯示正常進度圖示（未開始 / 已開始 / 已完成）
- 鎖定章節**可點擊**，點擊後顯示 Toast 提示：「請先完成前面的章節才能觀看此章節」
- Toast 類型：warning，顯示在頁面頂部，3 秒後自動消失
- 鎖定章節的文字顏色降低對比度（如 text-gray-400），視覺上區分已解鎖與被鎖定

### 鎖定章節頁面（直接 URL 存取被鎖定章節）
- 不顯示影片播放器
- 不顯示章節正文內容
- 顯示鎖定提示區塊（類似現有的 buy / 404 提示頁面佈局）
- 提示文字：「請先完成前面的章節才能觀看此章節」
- 可選：提供「前往下一個待完成章節」按鈕，引導學員回到正確位置

### 豁免對象
- 管理員（manage_woocommerce 權限）：不顯示任何鎖定狀態，所有章節正常存取
- 課程講師（teacher_ids）：不顯示任何鎖定狀態，所有章節正常存取
