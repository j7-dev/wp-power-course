# 章節內容頁

## 描述
教室頁面中的章節內容區塊，根據章節的影片類型和編輯器顯示對應內容。

## 關鍵元素
- 章節標題
- 影片播放器（根據 chapter_video.type 選擇播放方式）
- 章節文字內容（power-editor 或 elementor 渲染）
- 章節時長顯示（chapter_length 秒轉分鐘）
- PDF 文件（若有，帶浮水印）

## 影片自動續播
- VidStack 播放器接收 `video_progress_seconds` prop
- 影片載入完成（`onCanPlay`）後，若 `video_progress_seconds > 0` 且不超過影片 `duration`，自動 seek
- 若 `video_progress_seconds > duration`（影片被替換為較短版本），fallback 到 0 秒
- 支援影片類型：`bunny-stream-api`、`youtube`、`vimeo`
- 不支援影片類型：`code`（自訂 iframe，無法取得播放時間）、`none`（無影片）
