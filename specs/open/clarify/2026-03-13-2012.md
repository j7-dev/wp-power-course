# Clarify Session 2026-03-13 20:12

## Idea

我希望用戶在課程後台 /wp-admin/admin.php?page=power-course

使用 VideoInput， 選擇 Bunny Stream API 作為影片源時
可以多一個上傳 SubRip (SRT) files and Video Text Track (WebVTT) 字幕檔的選項
讓用戶從電腦上傳字幕檔

## Q&A

- Q: 字幕檔應上傳到哪裡？ → A: 上傳到 WordPress 媒體庫，前台播放時由 WordPress 提供字幕檔
- Q: 是否需要支援多語言字幕？ → A: 支援多語言（用戶可為同一影片上傳多份不同語言的字幕）
- Q: 用戶如何指定字幕的語言？ → A: 下拉選單選擇語言（預設常見語言列表，如 zh-TW、en、ja 等）
- Q: 用戶是否需要管理已上傳的字幕？ → A: 支援刪除與替換（列出已上傳的字幕，可個別刪除或重新上傳覆蓋）
- Q: SRT 和 WebVTT 兩種格式都要支援嗎？ → A: 兩種都接受上傳，後端自動將 SRT 轉為 WebVTT 儲存（前台統一用 WebVTT）
- Q: 前台 Vidstack 播放器是否需要顯示字幕切換功能？ → A: 是，使用 Vidstack 內建的字幕切換 UI（自動列出所有可用語言）
- Q: 字幕是否預設自動顯示？ → A: 預設關閉，學員需手動從播放器 UI 開啟
- Q: 是否需要在上傳前預覽字幕內容？ → A: 不需要，用戶上傳後直接到前台播放器確認效果
