# Issue #146: 紀錄用戶上次上到哪一個課程，以及幾分幾秒

## 需求摘要

為 Power Course 外掛新增影片播放進度記錄與跨課程「繼續上課」功能，
讓學員重新進入課程時可從上次中斷處繼續觀看。

## 已確認的設計決策

| # | 決策項目 | 選項 | 決定 |
|---|---------|------|------|
| 1 | 節流間隔 | 15 秒 | ✅ 15 秒 |
| 2 | 影片類型支援 | Bunny + YouTube + Vimeo | ✅ 不含 code（自訂 iframe） |
| 3 | UI 呈現 | 排序第一 + Badge | ✅ 最後課程排首位 + 繼續上課 Badge |
| 4 | 完成後進度 | 重置為 0 | ✅ 完成時歸零，回看從頭開始 |
| 5 | API 端點 | 獨立輕量端點 | ✅ `POST /chapters/{id}/video-progress` |
| 6 | 頁面關閉回報 | fetch + keepalive | ✅ 現代標準，支援自訂 headers |
| 7 | user_meta 更新時機 | 僅 visit_chapter hook | ✅ 不在高頻進度回報中更新 |
