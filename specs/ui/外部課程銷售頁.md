# 外部課程銷售頁 UI 規格

## 頁面結構

與站內課程銷售頁 (`specs/ui/課程銷售頁.md`) 佈局完全相同，以下僅列出差異項目。

## Header 區塊
- **特色影片**：若有設定 `feature_video`，正常顯示影片播放器；若無，顯示封面圖
- **講師資訊**：若有指派 `teacher_ids`，正常顯示講師頭像與名稱
- **徽章**：`is_popular`、`is_featured` 等徽章正常顯示
- **課程標題**：正常顯示 `name`
- **短描述**：正常顯示 `short_description`

## Body 區塊

### 課程資訊統計（info 區塊）
| 統計項 | 站內課程顯示值 | 外部課程顯示值 |
|--------|----------------|----------------|
| 開課時間 | wp_date 格式化 | `-` |
| 課程時長 | X 小時 Y 分鐘 | `-` |
| 章節數量 | N 個章節 | `-` |
| 觀看時間 | 限制描述文字 | `-` |
| 課程學員 | N 位學員 | `-` |

### 分頁內容（Tabs）
| Tab | 站內課程 | 外部課程 |
|-----|----------|----------|
| 課程介紹 | ✅ 顯示 | ✅ 顯示 |
| 章節列表 | ✅ 顯示 | ❌ 隱藏 |
| Q&A | ✅ 顯示（若啟用）| ✅ 顯示（若有 qa_list）|
| 留言 | ✅ 顯示（若啟用）| ❌ 隱藏 |
| 評價 | ✅ 顯示（若啟用）| ✅ 顯示（若啟用）|

## Sidebar 區塊

### 主商品卡片
- **價格顯示**：使用 WC 原生價格渲染（`regular_price` + `sale_price`），格式與站內課程一致
- **CTA 按鈕**：
  - 文字：`button_text`（預設「前往課程」）
  - 行為：`<a href="{product_url}" target="_blank" rel="noopener noreferrer">`
  - 樣式：與站內課程「立即報名」按鈕相同的主色調、尺寸
- **購物袋圖示按鈕**：隱藏（外部課程無購物車功能）
- **庫存資訊**：隱藏
- **已購學員數**：隱藏
- **倒數計時**：隱藏

### 銷售方案卡片
- **完全隱藏**：外部課程無銷售方案

## Mobile Fixed CTA
- 若 `enable_mobile_fixed_cta = yes`：
  - 顯示價格 + CTA 按鈕
  - CTA 按鈕文字與行為同 Sidebar
  - `target="_blank"` + `rel="noopener noreferrer"`

## Footer 區塊
- **試看影片**：若有 `trial_videos`（Issue #10），渲染規則與站內課程一致
  - 0 部：不渲染區塊
  - 1 部：直接顯示
  - 2~6 部：Swiper 輪播（無 autoplay、含分頁點與左右箭頭、切換時前一部自動暫停）
- **講師介紹**：若有 `teacher_ids`，正常顯示完整講師 bio

## 安全性
- `product_url` 輸出時須經過 `esc_url()` 處理
- 外部連結加上 `rel="noopener noreferrer"` 防止 window.opener 攻擊
- CTA `button_text` 輸出時須經過 `esc_html()` 處理
