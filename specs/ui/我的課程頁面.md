# 我的課程頁面

## 描述
WooCommerce My Account 頁面中的「我的課程」分頁，顯示學員已加入的所有課程卡片。
URL: `/my-account/courses/`

## 關鍵元素
- Tab 切換（所有課程 / 已開課 / 尚未開課）
- 課程卡片 Grid（3 欄響應式佈局）
- 課程卡片包含：課程封面、名稱、講師、進度條、觀看期限 Badge
- 「繼續上課」Badge 標示（最後造訪的課程）

## 繼續上課功能
- 系統根據 `pc_last_active_course` user_meta 判斷學員最後造訪的課程
- 最後造訪的課程排在列表第一位（在每個 tab 中）
- 最後造訪的課程卡片右上角顯示「繼續上課」Badge（取代原有的狀態 Badge）
- 若最後造訪的課程已到期，顯示「已到期」Badge（不顯示「繼續上課」）
- 無 `pc_last_active_course` 記錄時，維持預設排序，無「繼續上課」Badge

## 卡片連結導向
- 有 `last_visit_info` 記錄的課程：導向 `last_visit_info.chapter_id` 的教室頁面
- 無 `last_visit_info` 記錄的課程：導向第一個章節（按 menu_order 排序）
- 進入教室後，影片播放器根據 `video_progress_seconds` 自動 seek
