# 全域學員匯出按鈕

## 描述
後台全域學員管理頁面的「學員匯出 CSV」按鈕與確認 Modal，
允許管理員匯出跨越所有課程的學員名單 CSV。

## 位置
- 頁面：後台 → 學員管理（全域，非單一課程內的學員 Tab）
- 按鈕位置：頁面頂部操作列

## 互動流程
```
1. 管理員點擊「學員匯出 CSV」按鈕
2. 前端使用 useCustom hook 呼叫 GET /students/export-count（帶入當前篩選條件）
3. 彈出 Modal 確認對話框，顯示預估匯出筆數
4. 管理員點擊「確認匯出」
5. 前端開啟新視窗呼叫 GET /students/export-all（帶入相同篩選條件）
6. 瀏覽器自動下載 CSV 檔案
```

## 元素規格

### 匯出按鈕
- 文字：「學員匯出 CSV」
- 可見條件：管理員角色（manage_woocommerce capability）

### 確認 Modal
- 標題：確認匯出
- 內容：顯示「預估匯出 {count} 筆資料」
- 確認按鈕：「確認匯出」
  - 當 count = 0 時：按鈕 disabled
- 取消按鈕：「取消」

### 篩選條件傳遞
匯出 API 帶入與當前學員列表相同的篩選條件：
- `search`：搜尋關鍵字
- `avl_course_ids`：課程篩選
- `include`：指定用戶 ID

## 注意事項
- 前端使用 `useCustom` hook（Refine.dev），非直接使用 `AXIOS_INSTANCE`
- 匯出為新視窗下載，不影響當前頁面狀態
- 大量資料匯出時，後端使用 `batch_process` 分批處理避免記憶體溢位
