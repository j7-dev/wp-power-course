# 外部課程後台編輯頁 UI 規格

## 頁面結構

基於現有課程編輯頁（8 個頁籤），依課程類型動態調整。

## 課程類型 Radio（表單頂部，Tabs 上方）

### 新增模式
- 顯示 Radio Group：`站內課程` | `外部課程`
- 預設選中：`站內課程`
- 選擇後即時切換顯示的 Tabs 與欄位

### 編輯模式
- Radio Group 顯示對應的已選類型
- 狀態：`disabled`（不可切換，建立後不可轉換類型）

## 「課程描述」Tab — 外部課程額外欄位

在現有欄位（名稱、封面圖、描述、講師、特色影片等）之後，新增：

### 外部連結 URL
- **Label**：外部連結 URL
- **Type**：`Input`（URL 格式）
- **Placeholder**：`https://example.com/course`
- **驗證**：必填，必須以 `http://` 或 `https://` 開頭
- **對應欄位**：WC External `product_url`

### CTA 按鈕文字
- **Label**：CTA 按鈕文字
- **Type**：`Input`
- **Placeholder**：`前往課程`
- **驗證**：非必填，空值時前台顯示「前往課程」
- **對應欄位**：WC External `button_text`

## 「課程訂價」Tab — 外部課程版本

### 隱藏的欄位
- 產品類型選擇 Radio（簡單商品 / 定期定額）
- 訂閱相關所有欄位（訂閱價格、週期、試用期等）
- 庫存管理相關欄位

### 保留的欄位
- 原價（`regular_price`）— 標注「僅供展示」
- 特價（`sale_price`）— 標注「僅供展示」
- 特價日期範圍（`date_on_sale_from` / `date_on_sale_to`）

## 「其他設定」Tab

### 隱藏的欄位（外部課程不適用）
- 觀看期限設定（`limit_type`、`limit_value`、`limit_unit`）
- 開課時間排程（`course_schedule`）
- 教室連結（`classroom_link`）

### 保留的欄位
- 是否熱門（`is_popular`）
- 是否精選（`is_featured`）
- 啟用行動版固定 CTA（`enable_mobile_fixed_cta`）
- 自訂評價相關設定

## 「QA設定」Tab
- 完整保留，與站內課程相同
- 管理員可為外部課程設定常見問答

## 後台課程列表頁

### 新增按鈕
- 維持現有「新增課程」單一按鈕
- 點擊後進入編輯表單，由 Radio 選擇類型

### 列表欄位
- 無需新增欄位
- WC 產品類型欄位自動顯示「外部/加盟商品」作為識別

### 操作
- 外部課程支援：編輯、刪除
- 外部課程不支援：複製（因 WC External 類型複製邏輯需額外處理，暫不支援）
