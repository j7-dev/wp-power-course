# 外部課程需求釐清紀錄

## Issue: #155 - Power Course 支援外部課程

## 決策摘要

| # | 議題 | 決策 | 說明 |
|---|------|------|------|
| Q1 | 底層資料結構 | **(A) WC 內建 `external` 產品類型** | 使用 `WC_Product_External`，天然支援外部連結、`is_purchasable()` 回傳 `false` |
| Q2 | 卡片點擊行為 | **(A) 進入站內銷售頁** | 與站內課程一致，訪客先看完整介紹再決定 |
| Q3 | CTA 按鈕文字 | **(B) 管理員可自訂** | 每門外部課程可設不同文字，如「前往 Hahow」 |
| Q4 | 展示用價格 | **(A) WC 原生價格欄位** | 使用 `regular_price` / `sale_price`，卡片價格樣式與站內課程一致 |
| Q5 | 外部標記 | **(B) 卡片上有細微標記** | 右上角小外部連結圖示 ↗ |
| Q6 | 類型轉換 | **(A) 不需要轉換** | 建立後不可改變類型，需變更則刪除重建 |
| Q7 | 分類與標籤 | **(A) 共用相同系統** | 外部課程使用與站內課程相同的 `product_cat` / `product_tag` |

## 核心業務規則

1. **外部課程使用 WC `external` 產品類型** + `_is_course = 'yes'` meta flag
2. **不可購買**：`WC_Product_External::is_purchasable()` 天然回傳 `false`
3. **CTA 按鈕**：使用 WC `_button_text` 欄位，預設「前往課程」
4. **外部連結**：使用 WC `_product_url` 欄位，以 `target="_blank" rel="noopener noreferrer"` 開啟
5. **展示價格**：使用 WC 原生 `_regular_price` / `_price` 欄位
6. **功能隔離**：外部課程不參與章節管理、學員管理、銷售方案、自動郵件、自動授權、營收報表
